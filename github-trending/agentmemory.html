AgentMemory 为编程类智能体提供一个零外部依赖的持久化记忆层：12 个 Hook 自动捕获、4 层认知记忆模型、BM25 + 向量 + 知识图谱的三流融合检索。LongMemEval-S 基准 Recall@5 = 95.2%，年 token 成本约 $10。同时暴露 51 个 MCP 工具 与 107 个 REST 端点，是 mem0、Letta 之外"Agent 持久记忆"赛道第三极。

01项目概述

仓库 rohitg00/agentmemory 2026 年 5 月单日 +1,335 stars 登上 GitHub Trending Top 2。它解决的是一个反直觉的痛点：越聪明的 Coding Agent，越需要外部记忆——因为每个会话都是新的上下文窗口，项目架构、踩坑历史、用户偏好都要重新讲一遍。

"Your coding agent remembers everything. No more re-explaining."
— rohitg00 · AgentMemory README

它把这一层抽象成一个独立的记忆服务，对上提供 MCP 与 REST，对下用 SQLite 把状态、向量、图谱全部装下——理论上一个 npx 命令就能拉起。

02四层记忆模型

AgentMemory 仿照人类认知，把记忆切成四层。每层有不同的生命周期、检索粒度与衰减节奏；最底层的 Procedural 像"肌肉记忆"，最上层的 Working 像"当下的注意力"。

03三流融合检索

召回不是单一通路。AgentMemory 把关键词匹配、向量相似、知识图谱并行做，三股结果用 Reciprocal Rank Fusion (RRF) 重新排序——这是 LongMemEval 95.2% 召回率的工程基础。

NOTE · 隐私基线
内置隐私过滤器，写入前自动剥离 API Key 与 Secrets；默认本地 embedding（all-MiniLM-L6-v2），完全离线、$0 调用成本。REST 端点用 Bearer Token 鉴权。

07与同类项目的差异化

维度 AgentMemory mem0 · 53K ⭐ Letta · 22K ⭐ Agent 内置
Recall@5 95.2% 68.5% 83.2% N/A
自动捕获 12 个 Hook 手动调用 Agent 自编辑 手动
外部数据库 无（SQLite） Qdrant Postgres —
年 token 成本 ~$10 不一 核心放在上下文中 22K+ tokens
多 Agent MCP + REST + Lease API 仅 Letta 自身 各管各

四个关键差异

  1. 零外部依赖. 别人要一个向量库，它一个 SQLite 搞定。
  2. Hook 驱动自动捕获. 别人靠手动 API，它真正"无感"。
  3. 召回率显著领先. LongMemEval 95.2% vs. 68.5%。
  4. 真·多 Agent 友好. 同时暴露 MCP + REST，加上租约/信号机制做协调。

08如何快速上手

30 秒启动

npx @agentmemory/agentmemory

配置文件 ~/.agentmemory/.env

# LLM 提供方（可选；默认 no-op）
ANTHROPIC_API_KEY=sk-ant-...
GEMINI_API_KEY=...
OPENROUTER_API_KEY=...

# Embedding（自动检测；推荐本地）
EMBEDDING_PROVIDER=local

# 端口与特性开关
TOKEN_BUDGET=2000
AGENTMEMORY_SECRET=your-secret
AGENTMEMORY_AUTO_COMPRESS=false   # 默认关闭
AGENTMEMORY_SLOTS=false           # 可编辑记忆槽

在 Claude Code 中

启动后 Hook 会自动接管。也可手动调用 MCP 工具：

  • memory_recall — 召回相关记忆
  • memory_save — 主动保存
  • memory_smart_search — 混合语义 + 关键词检索
  • memory_sessions — 浏览会话

REST 调用示例

curl -X POST http://localhost:3113/agentmemory/smart-search \
  -H "Authorization: Bearer $AGENTMEMORY_SECRET" \
  -d '{"query": "auth middleware refactor"}'

本地开发

git clone https://github.com/rohitg00/agentmemory.git
cd agentmemory
npm install && npm run build
npm start          # 完整服务
npm run dev        # 热重载
npm test           # 800+ 测试

前置要求：Node.js ≥ 20 · iii-engine v0.11.2（或 Docker）
